Security information

The website does not store any user information, nor does it take any cookies, so there is no need to introduce encryption, as there is nothing to encrypt.

Upon entering information into the contact form, instead of sending it to a webserver, it will instead link the user to their email service, which is likely to be already secure. This way, the users data will not be able to be intercepted.
