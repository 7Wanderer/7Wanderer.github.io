Stylistic choices

The webpage was inspired by several band websites, including my own which I have made in the past. The text itself is formally written but relaxed in nature.

I picked my favourite colour to work as the 'main colour' of the website - a green backdrop. No colour combinations used should negatively affect people with deuteranopia.

RWD and mobile first

This website is designed to follow the 'mobile-first, responsive web design' philosophy by positioning each layer via grids. There is a separate section on the page dedicated to the header, main, pictures, footer and menu, and can reposition itself depending on the width of the device, using the @media CSS function.
